# 📈 Brazilian E-commerce Data Analysis & Power BI Dashboard

Welcome to the **Brazilian E-commerce Analysis** project! This repository provides a full data analysis pipeline following the CRISP-DM methodology, using the **Olist Brazilian E-commerce dataset**. The analysis uncovers key insights into sales performance, delivery efficiency, customer behavior, and product trends. An interactive **Power BI dashboard** is also developed to support business decision-making.

---

## 📌 Project Overview

This project applies the **CRISP-DM** process to analyze the Brazilian e-commerce market. Using Python for data wrangling and exploratory analysis, we derive insights to inform business strategies. The results are visualized in a **Power BI dashboard**, providing an interactive view of performance metrics.

---

---

## 🔍 CRISP-DM Steps

### 1. **Business Understanding**
- What are the key drivers of revenue?
- How efficient is the delivery process?
- Which sellers or products perform best?

### 2. **Data Understanding**
- Dataset Source: [Kaggle - Brazilian E-commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- Data includes: Orders, Products, Customers, Payments, Reviews, Sellers

### 3. **Data Preparation**
- Merged datasets for a unified view
- Handled missing values and duplicates
- Derived metrics like Order Value, Delivery Time, and On-time Delivery

### 4. **Exploratory Data Analysis (EDA)**
- Time series of orders and revenue
- Performance by product category and seller
- Review score distribution and delivery impact
- Payment methods and revenue contribution

### 5. **Modeling (optional)**
- No machine learning applied in this phase

### 6. **Evaluation & Insights**
- Operational bottlenecks and delivery delays identified
- High-value customers and top-performing products/sellers uncovered
- Most-used payment types and their business impact explored

### 7. **Deployment**
- Interactive **Power BI dashboard** designed for business use
- **Canva presentation** summarizing the analysis

---

## 📊 Power BI Dashboard Features

- **Overview:** Revenue trends, order volume, delivery performance
- **Financials:** Payment types, freight costs, revenue breakdown
- **Shipping:** On-time delivery rate, average delivery time by seller
- **Customers & Reviews:** Engagement levels, review score patterns
- **Products & Sellers:** Best-selling items, seller performance KPIs
